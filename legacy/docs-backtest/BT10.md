## Goal

Add annualization factor inference, DCA benchmark, subperiod/regime analysis, and fee sensitivity sweep.

## Deliverables

### New Modules

| Module | Class/Function | Description |
|--------|---------------|-------------|
| `engine.py` | `periods_per_year` parameter | Explicit annualization factor |
| `benchmark.py` | `dca_equity()` | Dollar-cost-average benchmark (auto/weekly/monthly) |
| `analyzer.py` | `BacktestAnalyzer` | Subperiod/regime/rolling/exit analysis |
| `fee_sweep.py` | `fee_sweep()` | Uniform fee rate sweep |
| `fee_sweep.py` | `maker_taker_sweep()` | Maker×Taker grid sweep |

### BacktestAnalyzer Capabilities

| Method | Purpose |
|--------|---------|
| `subperiod_metrics()` | Split equity curve by split_dates and compute subperiod metrics |
| `regime_conditional_metrics()` | Group by regime series and compute metrics |
| `rolling_metric()` | Rolling Sharpe/volatility/drawdown/return |
| `trade_analysis_by_exit()` | Trade statistics grouped by exit reason |

### Usage Example

```python

# Subperiod analysis
```
